## 설명

## Global Security 설정

|영역|옵션|설명|비고|
|---|---|---|---|
|Authentication|Disable remember me|사용자 로그인 정보를 기억하는 기능 제공 여부|default: enable|
|Authentication|Security Realm|사용자/그룹을 관리하는 방법|플러그인으로 다양하게 사용할 수 있음, 사용자 가입 허용을 해제하면 로그인 화면에서 회원가입 신청 버튼이 사라짐|
