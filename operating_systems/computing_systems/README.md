# Computing Systems
## 설명
컴퓨터는 on/off, 1/0 등과 같이 2가지만을 표현할 수 있는 2진 기법으로 모든 계산을 처리합니다.
이러한 컴퓨터를 이해하기 위해선 컴퓨팅체제를 이해하여야 합니다.

## 소개
운영체제의 기본이 되는 컴퓨팅체제를 이해하며 정리하는 페이지입니다.
